# VoiceMux Bridge ユーザーガイド 🛰️

VoiceMux Bridge（ボイスムクス・ブリッジ）へようこそ！
この拡張機能は、スマホの音声入力（Gboard や iPhone の音声入力など）を使って、パソコンのあらゆる入力欄（Gemini, ChatGPT, Gmail, Slackなど）へ文字を入力するための架け橋です。

難しい設定は必要ありません。このガイドと一緒に、最初の一歩を踏み出しましょう。

---

## 1. 準備：pairing session を同期する

まず、拡張機能を使える状態にします。

1.  パソコンで [pair.knc.jp/hub](https://pair.knc.jp/hub) または [VoiceMux Hub](https://hub.knc.jp) を開きます。
2.  ファーストパーティの pairing surface が開くと、現在の room 情報がこの拡張機能へ自動同期されます。
3.  ブラウザ右上の **VoiceMuxアイコン（マイクスタンドのマーク）** をクリックしてください。
4.  popup に **「スマホを接続」** または、すでに同期済みなら **「このPC上のWeb入力欄へ、スマホから音声入力できます」** と表示されていれば準備OKです。

> 💡 **ヒント**: アイコンをツールバーに「ピン留め」しておくと、いつでもすぐにQRコードを表示できて便利です。

---

## 2. つなげよう：スマホとペアリング

1.  表示された **QRコード** をスマホのカメラでスキャンします。
2.  スマホで専用の入力画面が開いたら、連携完了です。

あなたのスマホとパソコンは、強力な暗号（E2EE）で守られた専用の通り道でつながりました。

---

## 3. つかってみよう：いつものスマホで話しかける

VoiceMuxの最大の特徴は、スマホでいつも使っている **「Gboard」や「iPhoneの音声入力」** をそのままパソコンで使えることです。

1.  パソコンで、文字を入力したいサイト（ChatGPT, Gmailなど）の入力欄をクリックしてカーソルを合わせます。
2.  スマホの画面にある入力エリアをタップして、キーボードを出します。
3.  キーボードにある **「マイクのボタン」** を押して、スマホに向かって話しかけてみてください。
4.  話し終わると……あなたの言葉が、パソコンの入力欄に直接入力されます！

---

## 4. 必要なときだけ VoiceMuxHub を開く

VoiceMux Bridge の主機能は、PC 上の任意の Web 入力欄へ直接文字を入れることです。通常の音声入力のために VoiceMuxHub を開き続ける必要はありません。

VoiceMuxHub を開くのは、主に **校正したいとき** です。

---

## 5. 安心・安全のために

VoiceMuxは、あなたのプライバシーを一番大切にしています。
*   **あなたの文字は、誰にも見られません**: スマホとパソコンの間で直接暗号化されるため、途中のサーバーですら中身を読むことはできません。
*   **ローカル session cache**: 一度同期された room 情報は、room が reset または期限切れになるまで拡張機能のローカル保存領域から利用されます。

---

## 6. 困ったときは？

*   **popup が「スマホを接続」のまま**: [pair.knc.jp/hub](https://pair.knc.jp/hub) または [VoiceMux Hub](https://hub.knc.jp) をもう一度開き、現在の room を拡張機能へ同期してください。
*   **文字が入力されない**: パソコンの入力欄を一度クリックして、カーソルがチカチカしている状態にしてください。
