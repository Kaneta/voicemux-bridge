# 更新履歴 (Changelog)

VoiceMux Bridge の主要な変更履歴を記録します。

## [2.1.0] - 2026-02-21
### 追加
- **多言語対応 (i18n)**: 日本語と英語の表示に完全対応。
- **KNC ID ステートレス認証**: `hub.knc.jp` との自動セッション同期を実装。ユーザーによる手動設定なしでペアリングが可能に。
- **中央集権的復号ロジック**: セキュリティ向上のため、E2EE 復号処理を Background Script に集約。Content Script は平文のみを扱うよう簡素化。
- **Hub 連携ペアリング**: ポップアップの QR コードおよびリンク先を `hub.knc.jp` の統合 Zen UI へ変更。

## [1.6.1] - 2026-02-17
### 追加
- **透明性レポート**: 「なぜ全サイトへのアクセス権限が必要か」および「E2EE実装の技術的証明」ドキュメントを追加。
- **開発者プロフィール**: 信頼性向上のため、Google Play ストアのデベロッパープロフィールへのリンクを追加。
- **持続可能性と哲学**: 無料提供の理由と、AI開発による継続性についての説明を追加。

### 変更
- **クロスプラットフォーム互換性**: Goクライアントやモバイルとの鍵互換性を高めるため、Base64処理を改善。
- **安定性向上**: Manifest V3 環境下での WebSocket 接続を維持するため、定期的なハートビート処理を追加。
- **プロトコル統一**: 通信形式を Phoenix Protocol vsn=2.0.0 (Array 形式) に統一。

## [1.6.0] - 2026-02-15
### 追加
- **オンボーディングUI**: 初めてのユーザー向けに、ステップバイステップのガイド付きポップアップへ刷新。
- **Webモニターへのリンク**: 拡張機能内から v.knc.jp へ素早くアクセスできるリンクを追加。

### 変更
- **入力エンジンの刷新**: `InputEvent` を活用した新しい注入ロジックにより、Gmail, Gemini, ChatGPT 等の複雑なエディタへの対応を強化。
- **セキュリティ強化**: Origin検証の厳格化と、不要なパーミッションの削除。

## [1.5.0] - 2026-02-14
### 追加
- **マルチドメイン対応**: `v.knc.jp` (Zen) と `t.knc.jp` (翻訳) の両方に対応。
- **ドメイン移行**: 主要インフラを `knc.jp` ドメインへ統合。
